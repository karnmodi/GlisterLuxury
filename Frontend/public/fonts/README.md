# Fonts

This directory contains custom font files:

- **Web fonts**: WOFF, WOFF2 formats
- **Custom fonts**: Brand-specific typography
- **Icon fonts**: Font-based icon sets

## File Organization

- `primary/` - Primary brand fonts
- `secondary/` - Secondary fonts
- `icon-fonts/` - Icon font files
- `web-fonts/` - Web-optimized font files
