# Business Images

This directory contains business-related images:

- **Logo files**: SVG, PNG formats
- **Branding assets**: Company logos, brand elements
- **Marketing materials**: Banners, promotional images

## File Organization

- `logo.svg` - Main company logo (SVG format)
- `logo.png` - Main company logo (PNG format)
- `favicon.ico` - Website favicon
- `brand-assets/` - Additional branding materials
