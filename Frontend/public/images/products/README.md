# Product Images

This directory contains product-related images:

- **Product photos**: High-quality product images
- **Product catalogs**: Catalog images and thumbnails
- **Product variations**: Different angles, colors, finishes

## File Organization

- `bathroom-accessories/` - Bathroom accessory product images
- `mortise-handles/` - Mortise handle product images
- `thumbnails/` - Small thumbnail versions
- `high-res/` - High-resolution product images
