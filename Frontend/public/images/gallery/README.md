# Gallery Images

This directory contains gallery and showcase images:

- **Showroom images**: Physical showroom photos
- **Installation examples**: Real-world installations
- **Customer showcases**: Customer project photos
- **Lifestyle images**: Products in use

## File Organization

- `showroom/` - Showroom and display images
- `installations/` - Installation examples
- `customer-projects/` - Customer showcase projects
- `lifestyle/` - Lifestyle and usage images
